PAQUETE GITHUB PAGES - REPORTE aRMG

1. Descomprime este ZIP.
2. Entra a https://github.com.
3. Crea un repositorio llamado reportearmg.
4. En el repositorio, presiona Add file > Upload files.
5. Sube TODOS los archivos y carpetas del paquete descomprimido.
6. Presiona Commit changes.
7. Ve a Settings > Pages.
8. En Source selecciona Deploy from a branch.
9. Branch: main.
10. Folder: / root.
11. Presiona Save.
12. Espera 1 a 3 minutos.

Tu enlace será parecido a:
https://TU_USUARIO.github.io/reportearmg/

Prueba también:
/historial.html
/verificar.html
